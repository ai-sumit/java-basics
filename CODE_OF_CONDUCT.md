# Code of Conduct

## Our Pledge
We are committed to providing a welcoming and inclusive environment for everyone,
regardless of experience level, gender, identity, or background.

This project is intended to help learners grow together in a respectful way.

---

## Our Standards

Examples of positive behavior:
- Being respectful and kind
- Giving constructive feedback
- Helping beginners learn
- Respecting different opinions

Examples of unacceptable behavior:
- Harassment or hate speech
- Personal attacks or insults
- Trolling or spamming
- Disrespectful comments

---

## Enforcement Responsibilities
Project maintainers are responsible for clarifying and enforcing this Code of Conduct.
They may remove, edit, or reject contributions that violate these rules.

---

## Scope
This Code of Conduct applies to:
- GitHub issues
- Pull requests
- Discussions
- Comments and reviews

---

## Enforcement
If you experience unacceptable behavior, please report it by opening an issue
or contacting the repository maintainer.
